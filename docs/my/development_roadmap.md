# Дорожная карта разработки (REST API сервисы)

Оценки трудозатрат приведены как предварительные (для планирования) и требуют уточнения после согласования требований, контрактов и нефункциональных требований (нагрузка, SLA, безопасность).

| 3 | ToolCall Execution Service | Выполнение tool/function calls, нормализация результатов, маршрутизация на MCP/внутренние инструменты | Нужен для реальных действий ассистента и интеграций (инструменты/внешние операции) | 2 | 120 | 156 |
