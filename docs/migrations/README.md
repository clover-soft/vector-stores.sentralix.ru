# Миграции базы данных

Миграции хранятся в этой папке в виде SQL-скриптов.

## Как применять

1. Подключитесь к MariaDB/MySQL с правами на выполнение DDL.
2. Выберите нужную базу данных (например, `USE vector-stores;`).
3. Выполните миграции по порядку (по номеру в имени файла).

Пример:

```sql
SOURCE docs/migrations/0001_create_rag_files.sql;
```

## Правила

- Выполняйте миграции строго по порядку.
- Каждая миграция должна быть идемпотентной (использовать `IF NOT EXISTS`, где возможно).
